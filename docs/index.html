Willkommen

Dies ist eine einfache, statische Landing-Page. Die Seite ist für GitHub Pages mit einer eigenen Domain konfiguriert.

Du kannst diese Seite bearbeiten, indem du die Datei docs/index.html änderst und in den Branch main pusht. Änderungen werden automatisch auf GitHub Pages sichtbar, sobald die DNS- und GitHub Pages-Einstellungen korrekt sind.

  • Stil-Framework: Bulma (via CDN)
  • Domain: michelweiland.org

Letzte Änderung: siehe README.md

Bei Fragen zur DNS- oder Cloudflare-Konfiguration schreib mir kurz — ich helfe beim Feinschliff.
